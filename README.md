# random-ast

Generates random python 3.6 scripts

![random-ast screenshot](screenshot.png)

Resources:

* [ast module documentation](https://docs.python.org/3/library/ast.html)
* [better AST docs](https://greentreesnakes.readthedocs.io/en/latest/)
